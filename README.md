### 评语

1. 鸿蒙开发设计结合了Android，IOS，FLutter，RN等平台和方案的优点设计
2. TS(ArkTS)语言壁垒在鸿蒙这块降得很低，学习成本低，很好上手
3. 总得来说，非常nice，效率和舒适度比Android和Flutter高很多，内置控件和容器非常丰富，并且可定制化能力也很高
4. 未来可期，同时期待Flutter早日完美适配鸿蒙

### 后续目标
